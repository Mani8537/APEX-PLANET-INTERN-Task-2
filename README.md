# 📊 Exploratory Data Analysis (EDA) & Business Intelligence

## 📌 Project Overview
This project focuses on analyzing a sales dataset to uncover meaningful patterns, trends, and business insights. It combines EDA, SQL-based analysis, and dashboard visualization to support data-driven decision-making.

## 🎯 Objective
- Understand data distributions and patterns  
- Solve business problems using SQL queries  
- Identify relationships between variables  
- Build a dashboard to present key insights  

## 🛠️ Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)  
- SQL (MySQL)  
- Excel / Power BI / Dashboard Tools  
- Jupyter Notebook  

## 📂 Dataset Description
The dataset contains sales-related information, including:
- Product Name & Category  
- Order Date  
- Quantity Sold  
- Total Amount (Revenue)  
- Payment Method  
- Customer Ratings  
- City  

## 🔍 Step 1: Descriptive Statistics & Univariate Analysis
- Calculated key metrics like mean, median, and standard deviation  
- Used histograms and bar charts  
- Identified revenue distribution and customer patterns  

## 💡 Step 2: SQL for Business Questions
- Top 5 products by revenue  
- Monthly sales trend  
- Best-selling product category  
- Average order value  
- Most used payment method  
- City generating highest revenue  
- Highest rated product category  

## 📊 Step 3: Multivariate Analysis & Correlation
- Used scatter plots and heatmaps  
- Identified relationships between revenue, category, and ratings  

## 📈 Step 4: Dashboard Creation
Key KPIs:
- Total Revenue  
- Total Orders  
- Average Order Value  
- Total Customers  

Visual Insights:
- Revenue by Category  
- Monthly Sales Trends  
- Category Performance  
- Customer Trends  

## 📌 Key Insights
- Clothing & Electronics generate highest revenue  
- Sales peak in Q4 (holiday season)  
- Growth opportunities in underperforming categories  

## 🎥 Deliverables
- EDA Report (Jupyter Notebook)  
- SQL Queries with Results  
- Dashboard Presentation  
- LinkedIn Video  

## 🚀 Conclusion
This project shows how data can be transformed into actionable insights using EDA, SQL, and dashboards for business decision-making.
